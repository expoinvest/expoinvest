Русская Википедия определяет термин HTML так:

HTML (от англ. HyperText Markup Language - язык разметки гипертекста) - стандартный язык разметки докумеентов во всемирной паутине.

Пожалуй, ни убавить ни прибавить...

HTML позволяет формировать на Web-страницах следующие элементы:

  • абзацы;
  • заголовки;
  • цитаты;
  • списки;
  • таблицы;
  • графические изображения;
  • аудио и видео ролики.
